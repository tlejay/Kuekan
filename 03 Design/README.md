## Stitch Design Documents

**Primary design resources for KueKan platform using Google Stitch (https://stitch.withgoogle.com/)**

### Main Design Documents

1. **KueKan_Stitch_Design_Prompt_v1_2026-01.md**
   - Comprehensive design prompt for Stitch AI design system
   - Detailed specifications for all three applications (Customer, Partner, Admin)
   - Complete color system, typography, and component guidelines
   - Use this for detailed, professional design generation

2. **KueKan_Stitch_Quick_Start_Guide.md**
   - Fast-track guide for immediate design generation
   - 5 ready-to-use prompts for core screens
   - Iteration shortcuts and refinement examples
   - Perfect for rapid prototyping and quick iterations

3. **KueKan_Visual_Design_Reference.md**
   - Complete visual design system and brand guidelines
   - Color palette with hex codes and usage guidelines
   - Typography system with Thai and English fonts
   - Component library specifications
   - Cultural design considerations for Thai market
   - Design tokens for developer handoff

### How to Use These Documents

**For New Designers:**
1. Start with the Quick Start Guide
2. Copy the ready-made prompts into Stitch
3. Reference the Visual Design Reference for specific details

**For Detailed Work:**
1. Use the main Design Prompt document
2. Follow the screen-by-screen detailed prompts
3. Apply iterative refinement prompts as needed

**For Developers:**
1. Reference the Visual Design Reference
2. Use the Design Tokens section for implementation
3. Follow component specifications exactly

## Naming Convention
`[App]-[Screen]-[State]-[Version].[ext]`

Example: `Customer-MissionCenter-Active-v2.png`

---

## Design Workflow

1. **Generate in Stitch** using the prompts provided
2. **Export designs** at 2x resolution
3. **Organize in folders** following naming convention
4. **Document iterations** with version numbers
5. **Review with stakeholders** (PAO, NCC, DOS team)
6. **Handoff to development** with Visual Design Reference

---

## Key Design Principles for KueKan

1. **Vibrant & Engaging** - Encourage participation through gamification
2. **Trust-Building** - Enterprise-grade for government partnership
3. **Thai-First** - Cultural sensitivity and language optimization
4. **Mobile-First** - LINE-integrated experience
5. **Inclusive** - Accessible to all ages and tech literacy levels
