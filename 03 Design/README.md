# 03 Design

## Contents
- Wireframes
- UI mockups
- User flow diagrams
- Design system documentation
- Figma/Sketch file links

## Subfolders
- `/Customer` - Customer Mobile Site designs
- `/Partner` - Partner Mobile Site designs
- `/Admin` - Admin Console designs
- `/Shared` - Shared components and design system

## Naming Convention
`[App]-[Screen]-[State]-[Version].[ext]`

Example: `Customer-MissionCenter-Active-v2.png`
