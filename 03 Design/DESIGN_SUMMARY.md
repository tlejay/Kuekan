## Design System Quick Reference

### Colors (Primary)
```
Brand Teal:   #00B8A9 (Primary buttons, headers, active states)
Brand Gold:   #FFD93D (Point badges, rewards, achievements)
Accent Coral: #FF6B6B (CTAs, urgent actions)
Background:   #F8F9FA (Main app background)
```

### Typography (Thai)
```
Headings: Prompt (Google Fonts) - SemiBold, 24px-32px
Body:     Sarabun (Google Fonts) - Regular, 16px
Numbers:  Kanit (Google Fonts) - Bold, 20px-48px
```

### Key Measurements
```
Border Radius (Cards):    16px
Border Radius (Buttons):  24px (fully rounded)
Spacing (Base Unit):      4px
Card Shadow:              0 2px 8px rgba(0, 0, 0, 0.08)
Touch Target:             44x44px minimum
```

## Key Success Factors

### Design Must Achieve:

1. **Engagement** - Users feel motivated to complete missions
   - Target: 20% Daily Active Users
   - Vibrant colors and game-like interactions

2. **Trust** - Clear, official appearance
   - Enterprise-grade design quality
   - Government partnership credibility

3. **Accessibility** - Easy for all users
   - Thai language optimization (line-height 1.6)
   - Large touch targets (44x44px)
   - Clear visual hierarchy

4. **Cultural Fit** - Respects Thai culture
   - Thai patterns used respectfully
   - Local imagery (Korat landmarks)
   - Appropriate color symbolism

5. **Performance** - Fast and efficient
   - Optimized images
   - Minimal design weight
   - Quick loading times

## Common Prompts Quick Copy

### Make It More Thai
```
Add Thai cultural elements - use subtle ลายไทย patterns in the
background with 5% opacity and warm marketplace colors. Show Korat
landmarks in mission imagery.
```

### Improve Typography
```
Increase Thai text line-height to 1.6 for better readability. Use
Prompt font for headings and Sarabun for body text. Make numbers
bold using Kanit font.
```

### Add Animation
```
Add a celebratory coin animation when points are earned. Coins should
appear with scale animation (0 to 1.2 to 1.0) then fly to the pocket
icon in the header.
```

### Increase Engagement
```
Make mission cards more game-like with vibrant photography, progress
indicators, and prominent point reward badges with gold metallic gradient.
```

## Quality Checklist

Before finalizing any design:

**Visual Quality**
- [ ] Uses approved color palette
- [ ] Typography follows specs (Prompt/Sarabun/Kanit)
- [ ] Spacing follows 4px grid
- [ ] Border radius consistent (16px cards, 24px buttons)
- [ ] Shadows applied correctly

**Accessibility**
- [ ] Color contrast meets WCAG AA (4.5:1)
- [ ] Touch targets minimum 44x44px
- [ ] Focus states visible
- [ ] Thai text line-height 1.6+

**Thai Localization**
- [ ] Thai fonts render properly
- [ ] Cultural elements respectful
- [ ] Copy is friendly and clear

- [ ] Represents government partnership
- [ ] Appeals to Thai users
- [ ] Encourages participation
